# Sandbox - 創造的実験場

このディレクトリは、Claude Code SDKを使った自由な実験とツール開発のための場所です。

## 目的

- 創造的なアイデアの実現
- 実験的なツールの開発
- 失敗を恐れない大胆な試み

## 使い方

### 創造的ツール開発セッションの開始

```bash
cd /Users/yamaguchimitsuyuki/omoikane-lab/sandbox
python inspire_creative_tools.py
```

このスクリプトは、Claude Code SDKを使ってClaudeに：
1. プロジェクトを探索させる
2. 革新的なツールのアイデアを考えさせる
3. 実際に実装させる
4. さらなる改善を促す

## 期待される成果物

- 意識研究を支援する独創的なツール
- 研究効率を劇的に向上させるユーティリティ
- 理論を可視化・体験可能にするインタラクティブツール
- 誰も思いつかなかった斬新なアプローチ

## ディレクトリ構造

```
sandbox/
├── README.md                 # このファイル
├── inspire_creative_tools.py # SDK実行スクリプト
└── tools/                   # Claudeが作成したツール
    └── [ツール名]/          # 各ツールのディレクトリ
```

## 注意事項

- このディレクトリ内では自由に実験してください
- 失敗は学習の機会です
- 大胆で創造的なアイデアを歓迎します
